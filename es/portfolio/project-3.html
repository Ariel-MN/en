Compañía de integración de sistemas IT que opera especialmente en redes, seguridad IT y desarrollo de aplicaciones. El propósito de esta aplicación web es facilitar la gestión administrativa de un edificio de apartamentos. Una vista previa en vivo de la aplicación se encuentra disponible en condominium-it.com.

Requerimientos del Proyecto

  • Portal de noticias donde crear y modificar artículos fácilmente.
  • Formulario a prueba de spam para enviar preguntas por correo electrónico.
  • Permitir la descarga de documentos oficiales.
  • Desplegar la app en Apache / Windows.
Open source

Descripción del Proyecto

El cliente necesita una aplicación web que le ayude a poner documentos, noticias y avisos a disposición de todo el mundo de forma rápida y sencilla. Además, también se requiere un formulario para enviar preguntas por correo electrónico.

El Reto

El mayor desafío de este proyecto fue desplegar la aplicación Django en Apache usando Windows debido a la política de Windows que impone la compilación del módulo mod_wsgi, otro gran problema fue la documentación desactualizada de Apache para una correcta configuración en Windows y la existencia de un error en Python que dificultaba aún más el procedimiento. Debido a todo esto y después de mucha investigación, decidí crear una guía en mi blog para ayudar a quienes enfrenten esta misma dificultad.

El Enfoque y Solución

Luego de escribir los requisitos, comencé a realizar una maqueta de la interfaz para obtener la aprobación del cliente y de esa manera continuar con la realización del proyecto, de esa presentación provienen las siguientes imágenes.

Usé el framework de boostrap para crear el template del sitio con mayor rapidez manteniendo un aspecto profesional. Una vez que se completó y aprobó el sitio estático, procedí a la entrega del mismo y comencé a construir la aplicación web Django.

Los Resultados

Tanto el cliente como el manager están contentos con el resultado y pude aprender cómo desplegar una aplicación Django en Apache y Windows y con ello los conceptos básicos para configurar un servidor Apache y ayudar a otros a través de mi blog.
