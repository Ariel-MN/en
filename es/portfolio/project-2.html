Argumento: Hidden Folder

Un programa que oculta carpetas de manera eficiente de la interfaz del sistema operativo Windows.

Proyecto Personal

  • Descargar: Hidden-1.7-setup.exe
Proyecto personal que nació de mi deseo de esconder algunas carpetas cuando prestaba mi memoria USB a compañeros. La utilidad de este programa es ocultar efectivamente cualquier carpeta de la interfaz del sistema operativo Windows incluso en dispositivos USB, de modo que estas carpetas permanezcan ocultas en diferentes computadoras.

Requerimientos del Proyecto

  • Cuando la carpeta está oculta, todos los enlaces creados deben perder la conexión.
  • Mantener las carpetas ocultas incluso si el sistema operativo está configurado para mostrar archivos ocultos.
  • Las carpetas deben permanecer ocultas en memorias USB en otras computadoras sin instalar ningún software en ellas.
  • Consentir al usuario de elegir su contraseña para acceder a las carpetas ocultas por el programa.
  • Uso de bases de datos para almacenar la ubicación de las carpetas ocultas de cada usuario del programa y facilitar su visualización.
Open source

Descripción del Proyecto

Todos conocemos la función de ocultar del sistema operativo Windows.Hace su trabajo, si lo que quieres es que tu hijo de 6 años o tu abuelo de 80 no pueda encontrar tus archivos y arruinarlos. Pero a veces, esto no es suficiente y es necesario poder ocultar los archivos de una manera más sofisticada. Es posible que deba ocultarlos a sus compañeros de trabajo u otras personas que conozcan la mágica opción de "mostrar archivos ocultos" que se encuentra en algún lugar del explorador de archivos y la cual no revelaré en caso de que tengas menos de 6 años (risas).

Mucha gente deja la opción de ver los archivos ocultos habilitada de forma permanente. Por este motivo, es ineficaz ocultar archivos que no queremos que otros vean al prestarles una memoria USB. Con esto en mente, surge este proyecto, para satisfacer fácilmente esta necesidad.

El Reto

Exploré todas las funciones del sistema operativo Windows para encontrar una manera de ocultar mejor las cosas. Crear una forma en la que una vez oculta la carpeta de la GUI de Windows, poder ubicarla para volver a hacerla visible. Mantener un registro de las carpetas ocultas en caso de que el usuario olvide dónde están. Hacer una copia de seguridad de esta lista de ubicaciones de carpetas. Implementar la seguridad tanto del programa como de la base de datos utilizada para la copia de seguridad.

El Enfoque y Solución

Después de evaluar algunas opciones diferentes, elegí esta sencilla solución:Cree un programa con una interfaz intuitiva traducida a tres idiomas diferentes. Este programa cambia el nombre de la carpeta elegida para romper cualquier atajo adjunto a la misma y confiere una propiedad a la carpeta que indica que pertenece al sistema operativo, de esta manera el sistema resuelve ocultarla para evitar posibles daños no deseados a (lo que parece ser) sus propios archivos.

Los Resultados

¡Un instalador asistido que salva el día! Solo necesita ser instalado para ocultar las carpetas deseadas del usb o localmente y otras computadoras que usan Windows no podrán ver estas carpetas incluso si no tienen este programa instalado.
