Which programming language to learn first?

Published May 20201 min read

Python vs JavaScript

Why Python: Is an interpreted, high-level, object-oriented language, useful for backend, data science, artificial intelligence and more it can be used for build software from zero and also for scripting purposes. It is simple, practical and intuitive. Ideal to start learning. Has frameworks like Django and Flask.

Why JavaScript: It's a scripting language, useful for programming front-end, backend, databases, internet of things, robots and even servers. Simple and versatile. It comes installed by default in all internet browsers. It has frameworks like React and Angular.

I started learning with Python and later it was easier for me to learn JavaScript.

What was your first programming language? Which would you choose to learn between these two?
